# Online_Sales_Data_Anlysis (Interactive Dashboard creation using MS Excel)

## Project Objective
The objective of this project is to clean, transform, and analyze sales data spanning 2020 to 2025 using Excel Power Query, ensuring data accuracy and consistency across multiple years. By creating standardized formats and calculated metrics such as Gross Sales, Revenue, and Profit, the project aims to identify year-over-year trends, evaluate sales performance over time, and support strategic, data-driven business decisions.
